## 🇪🇸 Español

**Dungeon Terminal** es un motor de juego de rol (RPG) basado en texto desarrollado íntegramente en C++. El proyecto revive la esencia de los clásicos *dungeon crawlers* bajo una arquitectura moderna, centrada en la eficiencia de la memoria y la lógica procedural.

### 🚀 Características Principales
* **Sistema de Combate por Turnos:** Implementación de lógica de batalla con cálculos de atributos, estados y probabilidad de crítico.
* **Generación Procedural Ligera:** Estructuras de datos optimizadas para la creación de mazmorras y encuentros aleatorios.
* **Gestión de Inventario y Progresión:** Sistema dinámico de equipamiento, escalado de niveles y persistencia de estadísticas.
* **Arquitectura Limpia:** Aplicación de principios POO (Programación Orientada a Objetos) y manejo eficiente de memoria.

### 🛠️ Stack Tecnológico
* **Lenguaje:** C++ (Estándar C++17 o superior).
* **Paradigma:** Programación Orientada a Objetos (POO).
* **Herramientas:** G++ / CMake.

### 💻 Instalación y Ejecución
```bash
git clone [https://github.com/Hunkak03/Dungeon-Terminal.git](https://github.com/Hunkak03/Dungeon-Terminal.git)
cd Dungeon-Terminal
g++ -o dungeon_game src/*.cpp -Iinclude
./dungeon_game

🇺🇸 English
Dungeon Terminal is a text-based RPG engine developed entirely in C++. This project revives the essence of classic dungeon crawlers using a modern architecture focused on memory efficiency and procedural logic.

🚀 Key Features
Turn-Based Combat System: Battle logic implementation including attribute calculations, status effects, and crit chance.

Lightweight Procedural Generation: Optimized data structures for dungeon creation and random encounters.

Inventory & Progression Management: Dynamic equipment system, level scaling, and stat persistence.

Clean Architecture: Applied OOP (Object-Oriented Programming) principles and efficient memory management.

🛠️ Tech Stack
Language: C++ (Standard C++17 or higher).

Paradigm: Object-Oriented Programming (OOP).

Tools: G++ / CMake.
